Sets up the initial shape''s material by resolving, loading and decoding the given textures. The faceTextureIndices must match the geometry set with setGeometry(). If a texture can not be resolved, STATUS_UNABLE_TO_RESOLVE is returned and builtin:unknowntexture.png is used for that texture.

The sequence of setGeometry(), setUvs() and resolveTextures() calls does not matter, but some parameters mus be consistent.

Parameters
textureKeysThe textures (keys into texturesResolveMap / uris if resolveMap is 0). 0 or an empty string means 'no texture'.
textureCountThe number of textures.
faceTextureIndicesArray with texture index for each face.
facesCountThe number of faces (== faceCountsCount in setGeometry()). This is the number of elements in faceTextureIndices.
resolveMapResolveMap to use for texture lookups
cacheCache to use for texture lookups. 0 for no caching.
Returns
STATUS_OK on success or other detailed value in cause of failure.

Sets the initial shape's attributes. The major purpose is to override rule attributes with concrete values in order to control the procedural model generation. However, any attribute name can be used; this allows for passing initial shape specific attributes to the encoder. There is a number of reserved attribute names for special purposes; if present they influence the model generation: 

 L"streetWidth"                   - (float array) A semantic attribute to transport information about
                                    streets adjacent to the initial shape.
                                    If present it is used to evaluate the stret.xxx selectors in cga
                                    (for instance in the comp() operation). The index into the array
                                    corresponds to the edge-index, and the value corresponds to the
                                    width of the street on that side. 0 means no street, a value > 0
                                    means street.

 L"/cgaenv/"                      - Reserved namespace for controlling the cga generation.
 L"/cgaenv/occlDisableIntra"      - Disable intra-model occlusion queries (i.e. they all return false);
                                    bool; default: false
 L"/cgaenv/occlDisableInter"      - Disable inter-model occlusion queries (i.e. they all return false);
                                    bool; default: false
 L"/cgaenv/maxDerivationDepth"    - Maximum recursion level for cga operations;
                                    int; default: 99
 L"/cgaenv/maxDerivationWidth"    - Maximum shapetree width (for cga);
                                    int; default: 50000
 L"/cgaenv/maxFuncCallDepth"      - Maximum recursion level for cga functions;
                                    int; default: 256
 L"/cgaenv/floatCheck"            - Controls checking of float parameters of operation/built-in functions
                                    for inf/nan values:
                                     0 = ignore(don't check)
                                     1 = check and abort if inf/nan detected,
                                     2 = check and replace with 0  if inf/nan detected;
                                    int; default: 2
 L"/cgaenv/trimPlanesComputeSize" - The size of the computed trimplanes in meters;
                                    float; default: 100.0
 L"/cgaenv/touchesMaxDist",       - The maximum distance to distinct between touches and intersects;
                                    float; default: 1e-3f

 L"/cache/key"                    - Set the cache key associated to this initial shape. This allows e.g.
                                    for re-using an initial shape's cache key with a different instance.
                                    See also the generateOptions parameter of generate().
Parameters
ruleFileThe cgb file (as a key into the resolveMap).
startRuleThe start rule, with style prefix (e.g. "Default$Init"). This will be the initialShape.startRule cga attribute.
randomSeedThe random seed; this will be the seedian cga attribute of the initial shape.
nameThe name of the initial shape. This will be the initialShape.name cga attribute.
attributesThe rule attributes.
resolveMapThe resolve map to use in generate().
Returns
STATUS_OK on success or other detailed value in cause of failure.
